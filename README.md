# 2023 NYCU C programming final project

## Command line game
### GongZhu / Hearts / 拱豬

## Environment
### - **GNU/Linux 3.10.0-1160.95.1.el7.x86_64**
### - **gcc version 12.2.0**

## Download
### ```git clone https://github.com/Connection2Peter/NYCU-2023-C```

## Compile
### ```gcc GongZhu.c -o GongZhu```

## Play
### ```./GongZhu```

## Debug
### ```clear; gcc GongZhu.c -o GongZhu; ./GongZhu```
